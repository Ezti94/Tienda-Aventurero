# La Tienda del Aventurero

Proyecto web desarrollado con **HTML5 y CSS3** inspirado en los **Reinos Olvidados (Faerûn)** y en el universo de **Baldur's Gate**.

La página representa una tienda ficticia de objetos mágicos, armas, reliquias y suministros para aventureros. El objetivo del proyecto ha sido diseñar una web de una tienda de zapatillas, la prefería hacer temática para poder entretenerme mejor durante su creación intentando mantener una estructura clara, navegación sencilla y un estilo visual coherente en cada apartado.

## Descripción del proyecto

La web simula la tienda de un gremio de aventureros donde se pueden encontrar diferentes artículos relacionados con el mundo de fantasía de Faerûn dentro del juego Baldur's Gate 3.

El sitio incluye varias secciones que permiten:

- Explorar productos mágicos y equipo de aventura.
- Conocer la historia y sedes del gremio.
- Contactar con la tienda para realizar consultas o pedidos especiales.

---

## Estructura del proyecto

El proyecto está organizado en diferentes carpetas para separar correctamente los archivos HTML, CSS.

---

## Páginas del sitio

### Inicio
Presenta la tienda y muestra algunos elementos destacados de ella.  
Incluye información introductoria sobre los objetos de la tienda e información de Faerûn.

### Tienda
Catálogo de productos disponibles en la tienda del aventurero.  
Los objetos se presentan en formato de bloques tipo **cards**, mostrando imagen, descripción y precio.

### Nosotros
Describe la historia del gremio, sus sedes en distintas ciudades de Faerûn y algunas curiosidades sobre la tienda.

### Contacto
Página que permite al visitante enviar consultas al gremio mediante un formulario.  
También incluye información sobre la ubicación del establecimiento y redes del reino.

---

## Navegación

La navegación se realiza mediante el **menú principal del header**, disponible en todas las páginas del sitio.

Este menú permite acceder rápidamente al resto de secciones de la pagina web.

En dispositivos con pantallas pequeñas se utiliza un **menú hamburguesa** para mantener la navegación accesible.

## Información adicional

El formulario de contacto está diseñado y validado mediante HTML, pero no envía datos a una empresa real ya que el proyecto corresponde a una web inventada.

Las redes sociales presentes en la web son representaciones temáticas del mundo ficticio del proyecto, utilizando los recursos del **Font Awesome** con los iconos ya existentes de redes sociales ya existentes .
